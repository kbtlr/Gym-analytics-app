# Gym-analytics-app

An idea to allow a user to graphically and numerically track their gym stats in real time, such as weight loss, volume increase/decrease, and personal bests over a measure of time. This way, they have a visual measure of their progress, and can use this to influence future training plans.

A plain HTML+JS frontend with a Flask backend in Python. Runs in a web browser and uses Flask for HTTP comms and URL reading.

Initial focus on frontend, and producing a working entry screen and initial input. If frontend increases in complexity, may introduce React
